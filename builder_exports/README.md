# Builder Exports

This directory stores the working source copied from Frappe Builder and related website setup in Frappe Cloud.

Use it for:

- page `body.html` files copied from Builder
- page `data_script.py` files copied from Builder
- Web Form field lists and settings
- DocType field lists, naming rules, and notes

The live site still runs from Frappe Builder. This folder is the versioned copy.
